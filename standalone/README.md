# Projeto Final: Redes Neurais Aplicadas a Finanças (Hyena Hierarchy)

## Requisitos de Ambiente
Este projeto foi configurado para execução em ambiente Linux (testado em Ubuntu 24.04 / Fedora 42).

## 1. Instalação das dependências
É recomendável o uso de um ambiente virtual (venv) para evitar conflitos. No terminal, execute:

```bash
# Crie e ative o ambiente virtual
python3 -m venv venv
source venv/bin/activate

# Instale as dependências necessárias
pip install torch pandas scikit-learn
```

## 2. Executando o projeto no diretório
Execute o comando abaixo para executar o projeto:
```bash
python3 src/hyenafraud.py
```
